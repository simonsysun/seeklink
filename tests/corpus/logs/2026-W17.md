## 2026-W17 — Zettelkasten 几本书的交叉笔记

这周读完三本 zettelkasten 相关的：

1. **Sönke Ahrens, How to Take Smart Notes (2017)** — 最流行的入门
2. **Niklas Luhmann, Communicating with Slip Boxes (1981)** — Luhmann 自己写的方法论短文
3. **Bob Doto, A System for Writing (2023)** — 最近的实用派解读

### 共识

- 三本书都同意 [[atomic-notes]] 是核心。
- 三本书都警告**装饰型 vault** 的陷阱——漂亮的目录结构 ≠ 思考工具。
- 三本都说 permanent note 必须是**自己的话**。

### 分歧

- Ahrens 过度强调"结构化笔记流"（fleeting → literature → permanent），
  实际用起来很累，很多人卡在 literature 这步。
- Luhmann 原文根本没这三分法，他只分 Zettel 和 Verweisungszettel
  （卡片和索引卡片）。Ahrens 的三分是为了"教学便利"加上去的。
- Doto 的观点最有趣：**index 不重要**。如果你的链接做得够密，导航靠
  链接就行。Luhmann 用 index 是因为他没搜索。我们有搜索，所以
  可以砍掉 index。

### 对 seeklink 的启发

Doto 的观点等于说"**好搜索 = 不需要目录**"。这和 [[agent-memory-patterns]]
里的"retrieval quality = agent intelligence"是同一个 insight，只是
对 human 用户。

如果 seeklink 的搜索够准，你的 vault 可以扁平——logs/ notes/ cards/
三个文件夹就够，不需要 notes/learning-science/memory/ 这种层级。
准备把「扁平 vault + 强搜索」写成一张 permanent note。
