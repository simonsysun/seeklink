# 遗忘曲线 Forgetting curve

赫尔曼·艾宾浩斯（Hermann Ebbinghaus, 1850-1909）在 1880 年代用自己做受试者，
记忆了一批无意义音节（CVC 结构：WID、ZOF 等），然后在不同时间间隔后测量
还能回忆多少。他画出的曲线是心理学史上第一条严谨的实验曲线，后来被称为
**遗忘曲线**（forgetting curve）。

基本形状：记住一个新东西后，**最初的几个小时内遗忘最快**，一天丢掉一半左右；
一周后剩不到 25%；一个月只剩个位数。如果中间不复习，绝大多数学习是浪费的。

## 数学形式

Ebbinghaus 拟合的原始曲线是一个幂函数：

```
R(t) = 1 / (1 + (t / S) ^ c)
```

其中 `R(t)` 是时间 `t` 后的 retrievability（可回忆概率），`S` 是 stability
（保持期常数），`c` 是经验指数。现代模型（如 FSRS）用类似的幂律或指数衰减，
并让 `S` 随每次复习增长——这就是**间隔重复**的理论基础。

## 为什么这条曲线关键

- **实验复现**：后 140 年的研究反复验证了这条曲线的形状，跨语言、跨任务。
- **复习介入**：每次成功复习会重置并**延长 stability**。曲线不再是初次学习
  的陡坡，而是一系列越来越平缓的斜坡。
- **间隔算法**：任何间隔重复系统本质上都是在回答「下一次复习安排在
  retrievability 掉到多低时最优？」的问题。见 [[FSRS]] 的 90% 目标。

## 实践启示

- 学习后**当天晚上**做一次短复习比三天后多一次复习效果更好。
- 长间隔不等于懒复习——只要每次都"踩"在曲线下降的关键点。
- 课堂教学的一次性讲授，如果没有后续复习机制，从遗忘曲线看基本是浪费。
