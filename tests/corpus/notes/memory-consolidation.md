# 记忆巩固 Memory consolidation

学习到一个新东西之后，它最初存在海马体（hippocampus）的快速编码系统里。
如果不干预，这个痕迹几小时内就开始衰减（见 [[forgetting-curve]]）。**记忆
巩固**指的是大脑把短期痕迹"烧录"到长期皮层存储的过程——主要发生在睡眠
期间，尤其是慢波睡眠（SWS）和 REM 阶段。

## 关键发现

- **睡眠剥夺直接降低新记忆的保留率**。一整晚不睡后学的东西，一周后剩下
  的比正常睡眠少 30-40%。
- **巩固不是被动的重放**。睡眠中大脑会主动"重新体验"白天的关键片段，
  这个过程压缩了时间（几十毫秒 → 几秒），并把记忆和旧知识网络建立连接。
- **间隔重复 + 睡眠 = 最优组合**。[[spaced-repetition]] 的每次复习启动
  一次巩固窗口；睡眠让这次窗口真的生效。连续通宵刷 Anki 是反效果的。

## 对学习实践的启示

- **学重要的东西在睡前 1-2 小时**，让巩固期直接接上。
- **困了就停**。疲劳状态下的学习不进入长期记忆，徒增挫败感。
- **午睡（20 分钟）** 对当天学的新东西有测得到的巩固加成。
- **过度学习（overlearning）**——在刚会之后再多练 20%——能让巩固更稳。

## 和 FSRS 的关系

[[FSRS]] 不建模睡眠——它只建模时间。但实际使用中，规律睡眠的人能在
FSRS 的 90% 目标上稳定超出，睡眠紊乱的人会稳定低 5-10%。这不是算法的
锅，是生理的锅。
